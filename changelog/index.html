SecureClasses Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

Anything documented with RELEASED is a downloadable version from releases

The most recent Release​

V: [2.2.1] - 05-15-2026​

RELEASED​

Changes:​

  • Documentation updates to reflect new API: Opertable & Debug mode

Added:​

  • New property Opertable used to signify if the object has been destroyed
  • Debug mode during studio runtime

V: [2.1.3] - 05-08-2026​

RELEASED​

Changes:​

  • Documentation updates to reflect new __destroy limitations and behaviours

V: [2.1.0] - 05-08-2026​

Added:​

  • NewIndex protections in Metadata table

Changes:​

  • __destroy function run order
  • Better memory linking and utilization

V: [2.0.1] - 05-06-2026​

RELEASED​

Added:​

  • Documentation through Moonwave

V: [2.0.0] - 05-05-2026​

Removed:​

  • TransferMetatable Class
  • Userdata object support due to low use cases

Added:​

  • New Metamethod __typeKey
  • Runtime protections for _Private, __metamethod, ___Public, and Default property classes

Changed:​

  • Module returns function instead of dictionary
  • Temporarily removed type function from type cast

V: [1.0.0] - 04-12-2026​

RELEASED​

  • Initial release.
