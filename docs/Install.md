# Installation
# Current version: 2.0.1

## Method #1 - Roblox
Download straight from the [roblox marketplace](https://create.roblox.com/store/asset/100027805890469/SecureClasses) this will only contain the module itself.

## Method #2 - Github
Download straight from the releases subdirectory of this [repository](https://github.com/Green-Screen/SecureClasses/releases)

### Download options

    1. Install the [SecureClasses.V2.0.1WithDependencies.rbxm](https://github.com/Green-Screen/SecureClasses/releases/download/SecureClasses/SecureClasses.V2.0.1.WithDependencies.rbxm) file for just the runtime module with its dependencies
    2. Install the [SecureClasses.V2.0.1WithTests.rbxm](https://github.com/Green-Screen/SecureClasses/releases/download/SecureClasses/SecureClasses.V2.0.1.WithTests.rbxm) file for runtime and test scripts
    3. Install the [SecureClasses.V2.0.1.luau](https://github.com/Green-Screen/SecureClasses/releases/download/SecureClasses/SecureClasses.V2.0.1.luau) file WITHOUT dependencies. **[Dependencies](#dependencies) are needed to run**

## Method #3 - Git Clone
Clone this repository straight into your project with Git

### Git steps:

```bash
git clone https://github.com/Green-Screen/SecureClasses.git
```

## Dependencies:
-   [EasyType](https://github.com/Green-Screen/EasyType)
