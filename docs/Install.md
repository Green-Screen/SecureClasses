# Installation
# Current version: 2.1.3

## Method #1 - Roblox
Download straight from the [roblox marketplace](https://create.roblox.com/store/asset/100027805890469/SecureClasses) this will include dependencies and tests.

## Method #2 - Github
Download straight from the releases subdirectory of this [repository](https://github.com/Green-Screen/SecureClasses/releases)

### Download options

    1. Install the [V2.1.3 with dependencies](https://github.com/Green-Screen/SecureClasses/releases/download/V2.1.3/SecureClasses.V2.1.3.WithDependencies.rbxm) file for just the runtime module with its dependencies
    2. Install the [V2.1.3 with tests](https://github.com/Green-Screen/SecureClasses/releases/download/V2.1.3/SecureClasses.V2.1.3.WithTests.rbxm) file for runtime and test scripts
    3. Install the [V2.1.3 src](https://github.com/Green-Screen/SecureClasses/releases/download/V2.1.3/SecureClasses.V2.1.3.luau) file WITHOUT dependencies. **[Dependencies](#dependencies) are needed to run**

## Method #3 - Git Clone
Clone this repository straight into your project with Git

### Git steps:

```bash
git clone https://github.com/Green-Screen/SecureClasses.git
```

## Dependencies:
-   [EasyType](https://github.com/Green-Screen/EasyType)
